## 创建

`ng new ng-wyy --style=less --routing -S` // --routing 创建路由 -S 不要测试文件

## 依赖

1.ng-zero
`ng add ng-zorro-antd` // 前端 Ui

2.minireset.css
`npm i minreset.css`

## 创建模块

`ng g m core`

## Store 全局数据管理

`npm install @ngrx/store@8.6.0 --save`
`npm install @ngrx/store-devtools@8.3.0 --save`
